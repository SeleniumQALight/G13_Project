#TC-001 Successful user registration

    Test Data:
    username = imartyn;
    email = mailforivanna@gmail.com;
    password = 123456qwerty;

    Steps:
    1. Open Login page https://aqa-complexapp.onrender.com/
    2. Assert that Username field is present and empty on Login page
    3. Assert that Email field is present and empty on Login page
    4. Assert that Password field is present and empty on Login page
    5. Assert 'Sign up for OurApp' button is enabled on Login page
    6. Enter <username> into Username field on Login page
    7. Enter <email> into Email field on Login page
    8. Enter <password> into Password field on Login page
    9. Click 'Sign up for OurApp' button on Login page

    Expected Results:
    1. User is redirected to the page with latest posts (h2: Latest posts from those you follow)
    2. <username> is displayed on the top right corner of the page
    3. There's Create Post button
    3. There's Sign out button
    4. There's no Sign in button

    4. Assert that Password field is present and empty on Login page

    Expected Results:
