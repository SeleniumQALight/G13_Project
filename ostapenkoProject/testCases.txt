#001 Sign up new user

Preconditions:
* User is not logged in

Steps to reproduce:
1. Open the login page
2. Enter a valid username (≥ 3 characters) into the Username field
3. Enter a valid email (e.g., testUser@gmail.com) into the Email field
4. Enter a valid password (≥ 12 characters) into the Password field
5. Click the “Sign up for OurApp” button

Expected result:
* The “Sign Out” button is displayed
* The user’s username is displayed in the header

Postconditions:
* Delete the user from the database

#004 Create a private unique post

Preconditions:
* User is logged in
* Home page or dashboard is opened

Steps to reproduce:
1. Click the “Create Post” button in the header -> Post creation page opens
2. Enter a valid title into the Title field
3. Enter valid body content into the Body field
4. Check the “Is this post unique?” checkbox
5. Open the “Select value” dropdown and choose “Private”
6. Click the “Save New Post”  button

Expected result:
* The post is created successfully, confirmation message “New post successfully created.” is displayed
* The post is marked as unique
* The post is publicly visible
* The post displays the correct title and body content

Postconditions:
* Delete the post
