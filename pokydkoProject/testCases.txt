Pre-condition:
1. User is logged in

Steps:
1. Click on button "Create Post" on Home Page

3. Enter <postBody> into input Body Content on Create Post Page

Expected results:
1. I see success post creation message on Post Page
2. I see post title <postTitle> on Post Page
3. I see post body content <postBody> on Post Page
4. I see note with chosen value from dropdown on Post Page
5. I see that post is marked as unique on Post Page

Post-condition:
1. Delete the created post
2. Sign Out from the application
3. Close the browser
